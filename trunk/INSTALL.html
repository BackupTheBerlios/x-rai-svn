How to install X-Rai

B. Piwowarki

Last update: September 2007

Pre-requisite

In order to install x-rai, you need:

  • A web server (apache)
  • PHP with PEAR, modules: postgresql, xslt
  • a postgresql server
  • perl, bash

Installing

Decompress the archive or checkout via SVN. The web directory is the only one that have to be published by the web server, through a link for example with UNIX. The file structure must not be changed.

Change the name of "settings-template.inc" to "settings.inc" in web/include, and edit that file with the corresponding values.

The .xrai files

The .xrai files are used to describe the structure of a collection. The main document (the root of the structure) is the file index.xrai within the root directory of the x-rai file of the collection. They are used for allowing the user to browse into the collection. They are basically HTML elements with two special elements, namely subcollection and document.

<subcollection path="">...</subcollection>

<document path="">...</document>

A path must be a path to the corresponding x-rai in the case of "subcollection" or to an XML file of the collection in the case of "document".

The interface will create each time a link using as title text the text inside "subcollection" or "document".
